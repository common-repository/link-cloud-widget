=== Link-Cloud Widget ===
Contributors: pehbehbeh 
Tags: link, cloud
Requires at least: 2.8
Tested up to: 2.9.1
Stable tag: /tags/0.3/

Dieses Widget stellt Links in Form einer Cloud dar.

Die Links werden, je nach Bewertung, unterschiedlich gro&szlig; dargestellt.


== Installation ==
Die .zip-Datei entpacken und Inhalte in das wp-content/plugins/ Verzeichnis deiner WordPress Installation legen. Danach kann das Plugin im Backend aktiviert werden.
